WRF-Chem

  • In my PhD and work as an air pollution researcher, I've simulated air quality using a numerical weather prediction model (WRF-Chem) on a high-performance computer.
  • As there is little training online for this model, I've taught it to research staff and PhD students, and I've created and maintained an exhaustive user guide (here), which continues to be used for training.
  • More details can be found on the WRF-Chem at the University of Leeds GitHub.

Python

  • I use Python to analyse the large amounts of data produced by these complex models (GitHub).
  • I developed this website using Flask (with some HTML/CSS), and deployed it on GitHub Pages using Frozen-Flask.
  • I developed a Plot.ly Dash application to show how clean household energy can improve public health in India, and deployed it on PythonAnywhere here.
  • I have created a guide on how to merge high-frequency swaths from multiple satellites into a single high-resolution NetCDF. The example case is for MODIS aerosol optical depth from NASA, here.
